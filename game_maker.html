Termoxers Game Maker Guide

Start by making a Python File with your gameplay that will be ran in the Terminal.

Once you made your file, give it a unique name and make sure it ends with '.py'.

Make sure to make a github repository with all of the code and a 'README.md'.

And then send the code to termoxers@gmail.com.

If your file contains any libraries that aren't in the standard library then add a 'requirements.txt' file.

Your game will have to be approved by any Staff.

Hope to see your game!
